# Meta Reveal Maintenance Checklist

> The Meta Reveal chapter explicitly describes design decisions the reader experienced. If any of these change, the Meta Reveal must be updated to match. An inaccurate reveal breaks the standing ovation.

---

## Design Elements Referenced in the Meta Reveal

Each item below lists what the Meta Reveal currently says. If you change the design, update the corresponding paragraph in the Meta Reveal (in the DOCX, before About the Author).

### 1. The Cover
- **Current claim:** Soft-touch matte lamination, embossed/raised title, spot UV finish, hidden line of text on back cover visible only at certain angles
- **Update if:** Cover material, finish, or hidden element changes

### 2. The Color Arc
- **Current claim:** Parts 1-2 = cool steel blues ("you are learning"), Parts 3-4 = golds appearing ("you are applying"), Parts 6-7 = deep golds/ambers ("you have arrived")
- **Update if:** Color palette changes, Part structure changes, or the cool→warm progression is modified

### 3. The Typography
- **Current claim:** Garamond at 11pt on warm cream background. Processing fluency = trust (+40%). Geometric sans-serif chapter titles = announcement. Marginalia in lighter sans-serif = dual reading path (Greene model)
- **Update if:** Body font, chapter title font, or marginalia approach changes

### 4. The Chapter Openers (Hook Lines)
- **Current examples cited:** "Every person who walks toward you is already broadcasting." / "The boardroom is the most dangerous stage you will ever work." / "The walk to the stage is the performance."
- **Update if:** These specific hook lines change or are moved to different chapters

### 5. The Key Reads
- **Current examples cited:** "The read is never one signal. The read is the chain." / "The face is the performance. The hands are the truth." / "Tension is not the enemy. Boredom is."
- **Update if:** These specific key reads change

### 6. The Pattern Interrupts
- **Current claim:** Every 8-12 pages, layout changes. Dark pages, sidebar shifts, "What You Just Did" moments. Von Restorff effect.
- **Update if:** Frequency, style, or approach to pattern interrupts changes

### 7. The Margin Icons
- **Current claim:** Eye = BP, Profile silhouette = CR, Waveform = VS, Arrow = AM. Icons appear from Chapter 6 onward. By Chapter 10, readers process them unconsciously.
- **Update if:** Icon designs, categories, or introduction point changes. Also update chapter numbers if structure changes.

### 8. The Page Architecture
- **Current claim:** Key content on recto (right-hand) pages. Chapter closers end on recto. Key read = last thing in working memory before page turn.
- **Update if:** Recto bias approach changes

### 9. The Warm Cream Pages
- **Current claim:** Paper is warm cream, uncoated, slightly textured. Cream = craft, uncoated = authenticity.
- **Update if:** Paper stock changes

### 10. The Edge Colors
- **Current claim:** Fore-edge shifts from cool blue-gray (front) to deep warm gold (back). Seven shades, one per Part. Thumb index = progress bar.
- **Update if:** Edge printing approach or number of Parts changes

### 11. The "What You Just Did" Moments
- **Current claim:** Book breaks fourth wall — which hand holds the book, whether you skipped a heading, how long you've been reading.
- **Update if:** Specific "What You Just Did" moments change

### 12. The Summary List
The Meta Reveal ends with a rapid-fire list mapping design decisions to principles:
```
The gold accents — salience architecture.
The cool-to-warm color arc — behavioral priming.
The chapter hooks — serial position primacy.
The key reads — serial position recency.
The dark pages and shifted layouts — Von Restorff isolation.
The margin icons — repetitive priming and pattern recognition.
The marginalia — the Greene-model dual reading path.
The cream paper and readable fonts — processing fluency.
The recto page placement — attentional bias.
The page-turn suspense — open-loop tension.
The spot UV hidden text — the observation test.
The edge color gradient — progressive identity shift.
The "What You Just Did" moments — the mirror.
```
**Update if:** Any design element is added or removed. This list must match what the book actually does.

---

## How to Update the Meta Reveal

1. Open `Architecture-of-Wonder.docx`
2. Search for "THE META REVEAL" — it's between the Appendix and About the Author
3. Find the specific paragraph that references the changed element
4. Update the text to match the new design
5. If a design element was added, add a new paragraph describing it
6. Update the summary list at the end to include/remove the element

---

## The Final Three Lines

These should never change unless the entire book concept changes:

> *This book was not just written.*
>
> *It was designed to read you while you read it.*
>
> *And now you know how.*

These are set in gold, 14pt, with maximum surrounding whitespace. This is the standing ovation moment.
