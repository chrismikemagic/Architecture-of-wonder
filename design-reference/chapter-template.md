# The Architecture of Wonder — Chapter Template

> Use this template for every chapter. Consistency creates rhythm. The reader develops expectations that the book can satisfy — or strategically violate.

---

## Chapter Structure (In Order)

### 1. CHAPTER OPENER (Dark Page — Template A)

**Always recto. Always full dark gradient background.**

| Element | Content | Design |
|---------|---------|--------|
| Part indicator | e.g., "PART TWO" | Dim gray, 11pt sans, letterspaced, top of page |
| Chapter number | e.g., "06" | Gold, 72pt sans, light weight, centered |
| Chapter title | e.g., "BEHAVIORAL PROFILING IN REAL TIME" | White, 18pt sans, bold, all-caps, 5px letter-spacing |
| Hook line | One provocative sentence or question | Gray-blue italic, 13pt serif, below thin gold line |
| Tier badges | T1/T2/T3/T4 examples | Color-coded pills, centered |
| Margin icons | BP/CR/VS/AM with labels | Icons with abbreviated descriptions |

**The Hook Line (Serial Position — Primacy)**

This is the single most important sentence in the chapter. It must:
- Be provocative, not informative
- Create an open loop the reader wants closed
- Work in isolation (no context needed)
- Prime the reader's mindset for what follows

**Example hook lines:**
- Ch 1: *"Reality is not what happens. It is what they remember happening."*
- Ch 6: *"Every person who walks toward you is already broadcasting. The question is whether you are listening."*
- Ch 11: *"The boardroom is the most dangerous stage you will ever work."*
- Ch 16: *"The walk to the stage is the performance."*
- Ch 23: *"The booking was won or lost before you picked up the phone."*

---

### 2. EPIGRAPH (Optional — Top of First Body Page)

A quote from a relevant source that primes the reader's mindset via **associative priming**.

| Element | Design |
|---------|--------|
| Quote text | Serif italic, 11pt, gray-blue, centered |
| Attribution | Sans-serif light, 9pt, dim gray, right-aligned |
| Spacing | 24pt below epigraph before body text begins |

**Good epigraph sources:** Researchers cited in the chapter, historical figures, practitioners, unexpected sources that reframe the content.

---

### 3. DROP CAP (First Paragraph)

The first letter of the first paragraph is rendered as a drop cap.

| Element | Design |
|---------|--------|
| Letter | Sans-serif (Montserrat/Brandon Grotesque), Regular weight |
| Size | 3 lines tall |
| Color | Gold #C9A84C |
| Wrap | Body text wraps with 4pt clearance |

---

### 4. BODY TEXT (Main Content — Template B)

Standard body pages on warm cream background.

**Text specifications:**
- Font: Garamond or Minion Pro, 10.5–11pt
- Leading: 135%+ of type size
- Line length: 50–75 characters
- Color: Dark brown-black #2A2520
- Alignment: Left-aligned (ragged right) for optimal readability

**Running elements throughout body text:**

| Element | Frequency | Design |
|---------|-----------|--------|
| Margin icons (BP/CR/VS/AM) | Every observation reference | Icon zone, 60% opacity |
| Tier badges (T1–T4) | Every observation mention | Inline pills, color-coded |
| Section breaks | Between major sections | Gold centered dots (· · ·) |
| Section headers | As needed | Sans-serif bold, 12–13pt, gold underline |

---

### 5. MARGINALIA (Throughout — Greene Model)

**Outer margins only. Sans-serif light, 9pt.**

| Type | Color | Content Example |
|------|-------|-----------------|
| Field story | Gold #C9A84C | *"I once walked into a boardroom where..."* |
| Cross-reference | Steel Blue #1A8FA8 | *"See also #58 and #59 — these form a territorial cluster."* |
| Practitioner shortcut | Gold #C9A84C | *"In practice, skip to the cluster check first."* |
| Contrarian note | Gold #C9A84C | *"Some performers disagree — here's why that's fine."* |

**Rules:**
- Maximum 6 margin notes per chapter
- Align vertically with referenced paragraph
- Never longer than 3 sentences (use a sidebar for longer content)
- Gold for field content, Steel Blue for cross-references

---

### 6. SPOTLIGHT BOX (Once Per Chapter — Von Restorff)

**The ONE thing the reader will remember from this chapter.**

| Element | Design |
|---------|--------|
| Background | Dark gradient (#080F1A → #0D1E30) |
| Left border | 4px solid Gold |
| Corner radius | 4px |
| Label | "KEY PRINCIPLE" — Gold, sans-serif, 9pt, letterspaced |
| Content | White, serif italic, 12pt |
| Width | Full text block width |

**Content guidelines:**
- One sentence or short paragraph
- Should be quotable in isolation
- Captures the chapter's core insight
- If it wouldn't work as a pull quote or social media post, it's too long

---

### 7. PATTERN INTERRUPT (Every 8–12 Pages)

**Break the visual rhythm. Options:**

**Option A: Full-Bleed Dark Page (Template C)**
- Dark gradient background
- Single statistic, quote, or provocative statement in gold
- Supporting text in gray-blue italic
- Optional "What You Just Did" moment

**Option B: Sidebar Shift**
- Narrower column width with tinted background
- Different content type (case study, framework summary, exercise)
- Returns to standard layout on next page

**Option C: Pull Quote**
- Sans-serif, 14–16pt, gold
- Centered with gold rules above and below
- Maximum one per spread
- Preferably on recto page

**Option D: "What You Just Did" Moment**
- Thin gold vertical rule on left
- Italic serif, indented from body text
- Reveals something the reader just did unconsciously
- Maximum one per chapter

---

### 8. KEY READ (Chapter Closer — Serial Position Recency)

**The last element on the last page of every chapter. Preferably on a recto page.**

| Element | Design |
|---------|--------|
| Top rule | Gold, 0.5pt, 40% opacity, centered, 200px wide |
| Text | Gold, serif italic, 11pt, 600 weight, centered |
| Bottom rule | Gold, 0.5pt, 40% opacity, centered, 200px wide |
| Space above | Minimum 30pt |
| Space below | Minimum 20pt |

**Content guidelines:**
- One sentence maximum
- Must be memorable in complete isolation
- Should feel like a mic-drop or curtain line
- This is the serial position recency anchor — the last thing in working memory

**Example key reads:**
- Ch 6: *"The read is never one signal. The read is the chain."*
- Ch 14: *"The face is the performance. The hands are the truth."*
- Ch 16: *"The walk is the show. Everything after is confirmation."*
- Ch 22: *"You are not what you do. You are what they remember you doing."*

## Complete Chapter Checklist

For every chapter, confirm:

- [ ] Chapter opener on recto page with hook line
- [ ] Hook line is provocative, not informative (primacy)
- [ ] Drop cap on first body paragraph (gold, 3 lines)
- [ ] Epigraph present (if applicable) — primes mindset
- [ ] Body text at 50–75 characters per line
- [ ] Margin icons placed for every observation reference
- [ ] Tier badges inline for every observation mention
- [ ] Marginalia notes present (maximum 6, gold/blue, sans-serif light)
- [ ] ONE spotlight box per chapter (Von Restorff)
- [ ] Pattern interrupt every 8–12 pages
- [ ] Pull quotes on recto pages when possible (maximum one per spread)
- [ ] "What You Just Did" moment present (maximum one per chapter)
- [ ] Section breaks use gold centered dots
- [ ] Key read as final element (gold, italic, between rules) — recency
- [ ] Key read preferably on recto page
- [ ] Recto pages carry key content (recto bias)
